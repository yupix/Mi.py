API-Referenz#

In diesem Abschnitt beschreiben wir die API von Mi.py

Bemerkung

Diese Bibliothek kann debug mittels logging ausgeben. Sie sollten debug als Startargument übergeben, da es in der Standardkonfiguration nicht verfügbar ist.

Referenz des Ereignisses#

mi.on_ready(ws)#

Er wird aufgerufen, wenn der Bot die Verbindung zum Websocket aufgebaut hat.

mi.on_message(note: Note)#

Wird aufgerufen, wenn in dem Kanal, mit dem Sie verbunden sind, eine Notiz veröffentlicht wird

mi.on_emoji_add(emoji: Emoji)#

Wird aufgerufen, wenn ein Piktogramm zu einer Instanz hinzugefügt wird

mi.on_mention(mention: Note)#

メンションが含まれたノートなどが投稿された際に呼び出されます

Abstrakte Basisklasse#

Abstrakte Basisklasse <https://docs.python.org/ja/3/glossary.html#term-abstract-base-class>`_ ist eine Klasse, die geerbt werden kann, um eine Liste von Methoden usw. zu erhalten. Abstrakte Basisklassen können nicht instanziiert werden.
